# 地天泰

![upload.wikimedia.org/wikipedia/commons/1/13/Iching-hexagram-11.svg](https://upload.wikimedia.org/wikipedia/commons/1/13/Iching-hexagram-11.svg)

外至順內至剛，一生平安。

履之序卦，地天泰。履即有禮，禮之終極，始終如一，若所有人如此，治世來臨，
上地下天，天氣下降，地氣上升，陰陽交合。

外地內天，外順內健，外面謙和有禮，裏面實力十分充足，每日自強不息。
泰的時機到了，是因爲要處泰，必須有禮，始終如一。所以如果入社會無論到何處都有禮，始終如一，永遠都係泰。

治泰之道，爲包荒。指要有大器量，包容所有荒穢的人同事。譬如，蔣經國同麻風病人握手。

### 正名
地天泰，公司能否掛太太名字，不行。還行要掛先生名字。
如果家很大，西南西北都由房間，能否妻子住西南，丈夫住西北。可以，只要有宮。此時妻子居西北，公司可以掛妻子名字，但西南角一定要有宮。

西北角係陰陽調和嘅位置，一定要與夫同居。若獨居，或離婚或與丈夫死別，此時，母代父職，孤星到老。孤隂不生，獨陽不長。所以西北要夫妻同居，最好嘅時機就是結婚當晚開始入住。若嘈架後才去改，亦有效，本來就要離婚，改後三日到三個月內見效。

爲了結稅的問題，夫妻分別開兩個公司，就要分開瞓。若丈夫去西南與妻子同居，先生公司否。所以至少妻居夫位會好啲。
